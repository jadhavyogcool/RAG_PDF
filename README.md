# Deployment

To deploy this project run

Create Virtual Environment
```
python -m venv venv   
```
Activate Virtual Environment
```
# On Windows:
venv\Scripts\activate
# On Linux
source venv/bin/activate
```
Install all dependencies
```
pip install -r requirements.txt

```
freeze dependencies

```
pip freeze
```
